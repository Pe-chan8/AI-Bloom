# 🌻 AI-bloom ― 自分をやさしく抱きしめ、自信を育てる ―

## アプリ概要
**AI-bloom** は、日々の小さな頑張りを“やさしく抱きしめ”、
AIの問いかけで自己理解と自信形成を支える **セルフコンパッション × 自己分析ツール** です。

「今日起きられた」「話しかけてみた」「仕事で詰まったけど諦めなかった」  
そんな“誰にも言わないけれど、たしかに乗り越えた一歩”を短文で残すだけで、AIが

- 共感して肯定するメッセージ  
- 「なぜ？どうやって？どんな結果？」という深掘り質問  
- 強み抽出と自己PR文（STAR/CAR 形式）

を行い、**“自分のいちばんの味方”** となってくれます。  
婚活・就活・転職・人間関係など「評価される場」に出る前に、まずは自分を好きになる土台づくりをサポートします。

---

## 開発背景
**キャリアアドバイザーとして実務で感じた課題：**

1. **自己否定・他者比較の傾向**  
   小さな努力を見逃しがちな人ほど本当は努力家。ただ、他者評価を恐れて本音を話しにくい。  
   一方で「**AI なら本音を話せる**」という声も多かった。

2. **支援期間の限界**  
   対人支援は期間が限られるため、日常の細かな頑張りを拾い続けにくい。

**仮説：**

- **AI は本音を話しやすい相手になりうる。**  
- **日々の内省**が、評価される場での強さになる。 

“努力を言語化し、自信を形にする”  
“他者評価ではなく AI と一緒に自分を抱きしめる”

この **優しさと成長の循環** を実現するために、このアプリを作ります。

##  MVP 機能（第1フェーズ：**つぶやき → AI共感** に集中）
目標：2ヶ月で“癒される基本体験”を完成。機能を絞り、完成度と継続性を最優先。
### 1） つぶやき＆振り返りログ（CRUD）
1 日 1 行の「頑張り／大変だったこと」を投稿。  
- タグ付け・検索・並び替え  
- 心理的安全性を意識した UI（ポジ／ネガ歓迎）  
- **下書き／非公開デフォルト**・入力中オートセーブ

### 2） バディ AI による共感メッセージ
- 「あなたはちゃんと頑張ってる」方向の優しい応答（マイページにバディ常駐）  
- 感情トーンに合わせた**画像付きカード**生成（第2以降で本格化）  
- 保存・SNS シェア（X / Twitter）

### 3） ソーシャルタイプ診断（簡易）
自分の性格傾向を理解し、以後の対話トーンに活かします。

| タイプ | 特徴キーワード | 反応が響く要素 |
|---|---|---|
| Driving（ドライビング） | 行動・結果・決断 | 目標・達成・リーダーシップの承認 |
| Expressive（エクスプレッシブ） | 社交・感情・創造 | 共感・賞賛・共鳴 |
| Amiable（エミアブル） | 協調・思いやり・安定 | 優しさ・安心感・支え合い |
| Analytical（アナリティカル） | 論理・慎重・正確 | 理屈・整理・筋道立った理解 |

#### マイバディ機能（詳細）
トップページで「マイバディ（My Buddy）」が待っており、**「専属の小さなメンター」** として長期的に伴走。
選んだ **“口調と人格”** で対話が届く。  

| タイプ | 性格・話し方 | 支援スタイル | フレーズ例 |
|---|---|---|---|
| Analytical | 知的で落ち着く | ロジカルに整理し、根拠を言語化 | 「これは“積み重ねの結果”。根拠が見えますね。」 |
| Amiable | 優しく親しみやすい | 気持ちに寄り添い癒す | 「その優しさ、ちゃんと届いてるよ。無理しないでね。」 |
| Driving | 熱量高く率直 | ゴール明確化と一歩の後押し | 「今の一歩、最高！次はここを狙おう！」 |
| Expressive | 明るく感情豊か | 共鳴し自己表現を促す | 「それ、最高のストーリー！もっと聞きたいな！」 |

**相性表（既存図の維持）**  
| ユーザーのタイプ | 相性◎（最高の相棒） | 相性○（支え合える） | 相性△（刺激し合う可能性） | 相性▲（すれ違いやすい） |
| --- | --- | --- | --- | --- |
| Analytical（分析型）冷静・慎重・事実重視 | Analytical同じ論理型で安心感◎ | Amiableやさしく支えてくれる | Driving行動促進には良い刺激 | Expressive感情表現が多すぎて少し疲れるかも |
| Amiable（協調型）思いやり・安定志向 | Amiable安心できる相互支援関係 | Analytical冷静なサポートが落ち着く | Expressiveテンションに引っ張られるが元気が出る | Driving少し強引に感じることも |
| Driving（行動型）決断・達成志向 | Drivingテンポが合う仲間 | Expressive勢いを形にしてくれる | Analytical慎重さにイラッとする時もあるが学び多い | Amiableペースが合わず物足りない時も |
| Expressive（表現型）感情豊か・創造的 | Expressiveノリと感情で共鳴◎ | Driving勢いを形にしてくれる | Amiableやさしく受け止めてくれる | Analyticalロジカルすぎて噛み合わない時も |

## 第 2 フェーズ（本リリース：＋1 ヶ月）
### 1） 自己理解サポート
- つぶやき内容をもとに、**強み／ストレス傾向のレーダーチャート** を表示
- **連続投稿日数（Streak）・累計つぶやき数**
- **気分の傾向（ポジ/ネガ/ニュートラル）週次グラフ**

### 2） AI 深掘りアシスト（簡易版）
- 「なぜ？」「どうやって？」「結果は？」の 3 ステップ質問  
- 回答から **強みキーワード提示**  
- **STAR / CAR 形式の自己 PR 文を自動生成**

### 3） マイバディ拡張
- 毎週月曜：週替わりの共通コメント  
- 毎週日曜：**今週の総まとめ** コメント  
- 朝昼夜の挨拶  
- アニメーション・ミニイベント応答

### 4） モード切替 UI
- **癒しモード**（受容・抱擁重視）  
- **成長モード**（質問・分析重視）

### 5） バッジ機能
- 「今週のやさしさメダル」（投稿や自己配慮の達成で付与）で継続を促す

## 技術的配慮と工夫
| 項目 | 内容 |
|---|---|
| コスト管理 | 3.5優先、応答は要約キャッシュ／類似投稿キャッシュ（類似度 > 0.8 で再利用） |
| レート制限 | ユーザーごと：AI呼び出し 5回/時（MVP）。超過時は**事前生成メッセージ**を提示 |
| フォールバック | OpenAI 障害時：タイプ×気分の**テンプレ応答辞書**（ローカル）で即応 |
| オフライン | 入力はローカル保存→復帰時に同期（投稿消失ゼロ設計） |
| 安全性 | NGワード抑止／不適切応答の検知・握り潰し（検閲プロンプト） |
| A/Bテスト | **4タイプ×複数プロンプト**をローテーション→満足度/いいね率で最適化 |
| メトリクス | 応答時間・エラー率・トークン消費・レート制限ヒット率・**満足度**を計測 |
| プライバシー | 非公開デフォルト、個人情報の暗号化（AES-256）、自動バックアップ |

### セキュリティ対策
- 個人情報の暗号化（AES-256）
- APIキーの環境変数管理
- CSRF対策（Rails標準）
- SQLインジェクション対策（ActiveRecord）

## 差別化ポイント
### ① アプリ全体
- **共感 → 深掘り → 成長**：ほめて終わらず、**Streak** と **傾向グラフ**で内省を継続  
- **マイバディ AI**：自分に合うキャラが“相棒”として伴走  
- **ネガティブ受容**：マイナス感情も包む“抱擁アルゴリズム”  
- **自己承認設計**：他者評価ではなく“自分で認める”体験  
- **心理 × AI の UX**：癒しと分析を両立  
- **実務導線**：日常ログ→自己PR素材への橋渡し（第2フェーズ）

### ② ChatGPT との違い（差別化）
| 項目 | ChatGPT | AI-bloom |
|---|---|---|
| 対話目的 | 一般的な知識・質問回答 | 自己理解と自信形成に特化 |
| 応答トーン | フラット・論理的 | 共感・温かみ・安心感 |
| 構造 | 一問一答 | **記録→共感→深掘り→気づき** の内省ループ |
| キャラクター | なし | **AI「マイバディ」** が伴走 |
| 成果物 | 回答 | **自己 PR 文・強みリスト・自信ログ** |

ChatGPT が“問題解決の AI”なら、**AI-bloom** は“**自分を受け入れ、育てる AI**”。

## ユーザーテスト計画（MVP）
- 対象：5–10名（タイプ散らし）
- 時期：5週目（共感）実装後すぐ／7週目（深掘り）後に再実施
- 方法：タスクベース（「1つ投稿→応答を評価→また投稿したくなるか」）
- 収集：満足度（5段階）、“また使いたい”意図、自由記述
- 合否基準：満足度 4.0+、翌日再訪意図 60%+
